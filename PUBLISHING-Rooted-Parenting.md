# Publishing Rooted Parenting Online

Rooted Parenting is a static HTML, CSS, and JavaScript app. That means it can be published without a backend or database.

## Files to publish

Publish these files together:

- `index.html`
- `styles.css`
- `app.js`
- `parenting-support-app-content.json`
- `worksheets-printable.html`

## Easiest publishing options

### GitHub Pages

- Good if you want a free hosted website from a GitHub repository
- Best when you are comfortable uploading files to GitHub

### Netlify

- Good for drag-and-drop publishing
- Very beginner-friendly

### Vercel

- Good if you prefer publishing from GitHub with simple defaults

## Recommended beginner option: Netlify Drop

1. Open [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag your Rooted Parenting app files into the upload area
3. Wait for Netlify to publish the site
4. Open the live link and test the app

## After publishing, test these pages

- Home
- Behaviors
- Learn
- Tools
- Worksheets
- A few behavior guide pages
- A few lesson pages
- Printable worksheet page

## Helpful note

If you update the content later, republish the updated files so the live site stays current.
